# Ulangan_praktek_kenny_XIA

judul proyek= halaman evet
kodingan yang dibuat= halaman event cosplay

fitur:
* Informasi tentang event cosplay
* Beberapa cosplayer dan youtuber yang menghadiri acara
* website yang simple
* desain yang menarik
* pembelian e-ticket
